Introdução A Move.CSS é uma biblioteca de componentes web open source que disponibiliza código HTML, CSS e JavaScript para desenvolvimento de interfaces de usuário usando componentes com princípios básicos de animação.
Uso Para usar a Move.CSS é necessário importar a folha de estilo CSS através da tag link
<!-- Link para importar a versão 1 da Move.CSS --> <link rel="stylesheet" href="CSS_DA_MOVE_LINK">
Além da folha de estilo CSS é necessário importar também o arquivo JavaScript da biblioteca através da tag script. A Move.CSS utiliza o jQuery portanto é necessário importá-lo também.
<!-- Link para importar a versão 3.3.1 do jQuery --> <script src="https://code.jquery.com/jquery-3.3.1.slim.min.js"></script> <!-- Link para importar a versão 1 da Move.CSS --> <script src="JS_DA_MOVE_LINK"></script>
Template Básico A seguir um exemplo de um arquivo HTML com os arquivos necessários para o uso da biblioteca importados
<!doctype html> <html lang="en"> <head> <meta charset="utf-8"> <meta name="viewport" content="width=device-width, initial-scale=1, shrink-to-fit=no"> <!-- Move.CSS --> <link rel="stylesheet" href="CSS_DA_MOVE_LINK"> <title>Hello, world!</title> </head> <body> <h1>Hello, world!</h1> <!-- jQuery --> <script src="https://code.jquery.com/jquery-3.3.1.slim.min.js"></script> <!-- Move.CSS --> <script src="JS_DA_MOVE_LINK"></script> </body> </html>
Download Além do import através dos links você também pode baixar os arquivos da biblioteca e tê-los localmente. Na pasta você encontra-rá os seguintes arquivos.
movecss/ ---- css/ -------- move.min.css -------- move.css ---- js/ -------- move.min.js -------- move.js
